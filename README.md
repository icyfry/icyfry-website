## Devs Commands

### Setup

```shell
npm install -g @vue/cli @vue/cli-service-global
```

### Compiles and minifies for production

```shell
npm run build:prod
npm run build:staging
```

### Run tests

```shell
npm run test:unit
npm run test:e2e
```

### Compiles and hot-reloads for development

```shell
npm run serve
```
