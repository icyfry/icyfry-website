## Devs Commands

### Setup
```
npm install -g @vue/cli @vue/cli-service-global
```

### Compiles and minifies for production
```
npm run build-prod
```

### Run tests
```
npm run test
```

### Compiles and hot-reloads for development
```
npm run serve
```
